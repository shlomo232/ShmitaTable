To download calendar, click the button.

This is an ICAL calendar file that can be added to any calendar application (Outlook, Google Calendar, etc.). It will then alert you whenever the shmita status of any food changes.

How to use:

To use this calendar, follow these steps:

  1. Create a new calendar. (This is recommended, not required. If you ever want to delete the Shmita Table calendar entries, you can just delete this calendar without affecting any events that were previously in your calendar.)

    In Google Calendar, this is done by going to the bottom left of the calendar, and choosing My calendars->Create new calendar.

  2. Import the Shmita Table file to this calendar.

    In Google Calendar, this is done by going to the bottom left of the calendar, and choosing Other calendars->Import calendar.

    Then, select the file you downloaded, and choose the new calendar you just created.

כדי להוריד את יומן, יש ללחוץ על הכפתור:

זה קובץ ICAL שניתן להוסיף אותו לכל אפליקציית לוח שנה (Outlook, Google Calendar וכו'), והוא יודיע לך על כל שינוי סטטוס של מאכל בשנת השמיטה.

איך משתמשים:

כדי להשתמש ביומן, יש לבצע את הפעולות הבאות:

  1. ליצור יומן חדש. מומלץ ולא חובה. אם אי פעם תרצה למחוק את יומן השמיטה מהיומן שלך, רצוי שיהיה יומן נפרד לשמיטה כדי שתוכל למחוק רק אותו בלי להשפיע על שאר האירועים שהיו ביומן שלך.

    ביומן גוגל, יש ליצור יומן על ידי בחירת היומנים שלי - צור יומן חדש בצד הימני של היומן.

  2. יבא את הקובץ ליומן זה.

    ביומן גוגל, יש לבחור יומנים אחרים - יבא יומן.

    אז, תבחר את הקובץ שהורדת ואת היומן שכעת יצרת.
